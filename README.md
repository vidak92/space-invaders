## About
- This is a sample game project created with [Unity](https://unity.com/)
- It's a simplified version of the classic [Space Invaders](https://en.wikipedia.org/wiki/Space_Invaders) game
- Developed for WebGL, works on desktop and mobile platforms
- Check out the playable build [here](https://vidak92.github.io/space-invaders-build/)
- [SGS Tools](https://github.com/vidak92/sgs-tools) used as a custom dependency (utility-style library for Unity)
- Assets by [Kenney](https://www.kenney.nl)
